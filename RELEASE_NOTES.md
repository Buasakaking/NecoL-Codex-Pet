# NecoL codex宠物 v1.0.0

## 包含内容

- 睁眼/闭眼双帧待机动画。
- 工作与等待状态使用完整六帧吃薯片动画。
- 动画持续循环到任务状态结束。
- 失败动画与左右拖拽动画。
- PowerShell 一键安装器和 Codex 自动安装协议。

## 部署方式

完整部署方案、版本检查、校验和、启动路径、验证标准和回滚方式见仓库根目录的 `CODEX_INSTALL.md`。

手动安装：下载 `NecoL-codex-pet.zip`，完整解压后运行 `install.ps1`。
